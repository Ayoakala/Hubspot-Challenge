# Hubspot-Challenge
Spring MVC REST API

Export a project into Eclipse or STS as a Maven project
Start the Tomcat Server
Run the application on tomcat server
Use the utilities for testing the application by sending JSON data, I have used Postman
URL : localhost:8080/hubspot-api/partners
If your answer is correct, the API will return 200 OK. If the request is malformatted or incorrect, 
the API will return 400 along with a message indicating if the response is of the wrong structure or incorrect.
